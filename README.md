# DataValidation
It has two python files
1. Classes count graph which will create the graphs based on the count of each class annotated in the dataset
2. Combinations split - This script is used to split the classes based on the combinations so that we can easly find anomaly in the dataset.

How to run:

If you are on windows run the "DataValidation.bat" file

if you are on linux based system "sorry guys you need to do it manully" run the two scripty manually "python classes_count_graph.py" and "python Image_TXT_Combinations_Seperate.py"
